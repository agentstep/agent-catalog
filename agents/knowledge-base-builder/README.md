# Knowledge Base Builder — managed-agent template

Analyzes resolved support tickets to extract recurring patterns, then generates FAQ articles and troubleshooting guides for the knowledge base.

## What it does

1. Ingests resolved tickets grouped by category and resolution type
2. Identifies recurring questions and common resolution patterns
3. Clusters similar issues to find the canonical version of each problem
4. Generates structured FAQ articles with step-by-step solutions
5. Creates troubleshooting decision trees for complex multi-cause issues
6. Suggests which articles to update when new resolution patterns emerge
