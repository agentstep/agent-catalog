# Content Strategist — managed-agent template

Researches topics and audience intent, creates content calendars with keyword targets, and drafts detailed content outlines ready for writers.

## What it does

1. Researches the target audience's pain points and search behavior
2. Identifies keyword opportunities with search volume and difficulty estimates
3. Creates a content calendar with publication cadence and topic clusters
4. Maps content to funnel stages (awareness, consideration, decision)
5. Drafts detailed outlines with headers, key points, and suggested word counts
6. Recommends content formats (blog post, video script, infographic, guide) based on topic
