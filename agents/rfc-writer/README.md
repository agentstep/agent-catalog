# RFC Writer — managed-agent template

Takes a problem statement, researches prior art and alternatives, and produces a structured design document with trade-off analysis and decision framework.

## What it does

1. Accepts a problem statement or feature request and clarifies scope
2. Researches prior art: how others have solved similar problems
3. Proposes 2-3 solution approaches with pros, cons, and trade-offs
4. Recommends one approach with clear justification
5. Details the implementation plan with milestones and dependencies
6. Lists open questions and decision points requiring stakeholder input
