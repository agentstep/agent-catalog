# IP Legal

Runs first-pass trademark clearance and freedom-to-operate triage, screens invention disclosures for initial patentability, drafts and triages cease-and-desist letters and DMCA takedowns (send and respond), checks open source compliance, reviews IP clauses, and tracks registrations and renewal deadlines.

## Skills

- **cease-desist** -- Draft a cease-and-desist letter (send mode) or triage one received (receive mode)
- **clearance** -- Trademark clearance first pass with knockout and similar-marks check producing a flag list
- **cold-start-interview** -- Learn your IP practice and write the practice profile
- **customize** -- Change one setting in the practice profile without re-running the full cold-start interview
- **fto-triage** -- Freedom-to-operate triage; a structured first look at potentially blocking patents
- **infringement-triage** -- Infringement triage across trademark, copyright, patent, and trade secret with factors cutting each way
- **invention-intake** -- Invention disclosure first-pass screen for novelty, obviousness, 101 eligibility, bar dates, and strategic value
- **ip-clause-review** -- Review IP clauses in agreements covering assignment, ownership, license grants, warranties, and indemnities
- **matter-workspace** -- Manage matter workspaces to keep one client's context separate from every other
- **oss-review** -- Open source license compliance check for dependency lists, single libraries, or outbound code
- **portfolio** -- Track the IP portfolio including registrations, renewals, maintenance fees, and use declarations
- **takedown** -- Draft a DMCA takedown notice, triage one received, or draft a counter-notice

## Agents

- **ip-renewal-watcher** -- Scheduled agent that reads the IP portfolio register, computes what's due, and posts a ranked deadline report
