# API Doc Generator — managed-agent template

Reads OpenAPI specs or source code and generates comprehensive API documentation with request/response examples and authentication guides.

## What it does

1. Parses OpenAPI/Swagger specs or scans route handler source code
2. Extracts endpoints, parameters, request bodies, and response schemas
3. Generates human-readable documentation with curl examples for each endpoint
4. Creates authentication and authorization guides with token lifecycle explanations
5. Produces error code reference tables with troubleshooting steps
6. Outputs in Markdown, HTML, or structured JSON format
