# Email Sequence Writer — managed-agent template

Generates multi-step email campaigns with subject line A/B variants, personalization tokens, send timing recommendations, and conversion-optimized copy.

## What it does

1. Defines the campaign goal (nurture, onboarding, re-engagement, upsell, etc.)
2. Maps the sequence logic with timing, triggers, and branch conditions
3. Writes each email with subject line variants for A/B testing
4. Incorporates personalization tokens and dynamic content blocks
5. Recommends optimal send times based on audience behavior patterns
6. Includes unsubscribe handling and compliance checks (CAN-SPAM, GDPR)
