# Draft Employer Compliance Certification — managed-agent template

Drafts a comprehensive Employer Compliance Certification for USCIS in response to a Fraud Detection and National Security (FDNS) site visit and associated Request for Evidence (RFE). Addresses the employer's bona fide job offer, worksite conditions, wage compliance, and supervisory structure.

## What it does

1. Reads the FDNS site visit report or RFE letter to identify specific concerns
2. Reads employer information, organizational documents, and payroll data
3. Structures a certification addressing each FDNS/RFE concern point by point
4. Includes supporting attestations regarding job duties, wages, worksite, and supervision
5. Drafts recommended exhibits list with supporting documentation
6. Produces the certification letter and exhibits checklist

## Inputs

- FDNS site visit report or RFE letter from USCIS
- Employer organizational information (company description, org chart, financial statements)
- Beneficiary employment details (job title, duties, salary, worksite, supervisor)
- Payroll records, tax filings, or other wage verification documents

## Outputs

- `./out/employer-compliance-certification.md` — the certification letter
- `./out/certification-exhibits-checklist.md` — recommended supporting documents
