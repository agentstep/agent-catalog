# Onboarding Coordinator — managed-agent template

Generates role-based onboarding checklists, answers handbook questions, schedules key meetings, and escalates blockers to the appropriate team.

## What it does

1. Takes the new hire's role, team, and start date as input
2. Generates a customized onboarding checklist with day-by-day tasks for the first 30/60/90 days
3. Answers questions about company policies, benefits, and processes from the handbook
4. Identifies required access provisioning (tools, repos, environments) and tracks completion
5. Schedules introductory meetings with key stakeholders
6. Escalates blockers (missing access, unclear ownership) to the appropriate team lead
