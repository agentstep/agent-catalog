# Draft Response to Notice of Inspection — managed-agent template

Prepares a response package to an ICE Homeland Security Investigations (HSI) Notice of Inspection (NOI) concerning I-9 records and employment eligibility verification. Ensures the employer produces all required documents within the statutory timeframe while preserving legal rights and mitigating penalty exposure.

## What it does

1. Reads the NOI and identifies all specific document requests
2. Analyzes the scope of the inspection (full workforce or targeted)
3. Prepares a response strategy addressing each requested category
4. Drafts the cover letter and document production log
5. Identifies I-9 correction opportunities within the good-faith window
6. Produces the response package and a compliance action plan

## Inputs

- ICE Notice of Inspection (NOI) document
- Employer's I-9 records and employee roster
- Payroll records, business licenses, and other documents specified in the NOI
- Prior inspection history (if any)

## Outputs

- `./out/noi-response-letter.md` — cover letter and production log
- `./out/noi-action-plan.md` — pre-production compliance steps and post-production strategy
